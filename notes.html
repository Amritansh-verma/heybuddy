Java Notes

            
            1. Java Overview
            - Java  ek high-level, object-oriented programming language hai, jo 1995 mein Sun Microsystems (Oracle) dwara launch ki gayi thi.
            - Java ko "write once, run anywhere" ke principle par design kiya gaya hai, jo iska portability feature dikhata hai.
            
            
             2. Java Development Environment
            - Java Development Kit (JDK): Java programming ke liye zaroori tools ka set hota hai, jisme Java compiler aur runtime environment shamil hote hain.
            -Java Runtime Environment (JRE): Yeh Java applications ko run karne ke liye required environment provide karta hai.
            - Integrated Development Environment (IDE): Popular IDEs hain Eclipse, IntelliJ IDEA, aur NetBeans, jo code likhne aur debugging karne mein madad karte hain.
            
            
             3. Basic Syntax
            
             Hello World Program java
            public class HelloWorld {
                public static void main(String[] args) {
                    System.out.println("Hello, World!");
                }
            }
            
            
            public class HelloWorld : Class declaration. Java mein sab kuch classes ke andar hota hai.
            
            public static void main(String[] args): Main method, jo program execution ka entry point hota hai.
            
            System.out.println(): Console par output display karne ke liye use hota hai.
            
            
             4. Data Types
            Java mein 2 tarah ke data types hote hain:
            
            
             a. Primitive Data Types
            
            int: Integer values ke liye (4 bytes).
            
            double: Decimal values ke liye (8 bytes).
            
            char: Single characters ke liye (2 bytes).
            
            boolean: true ya false values ke liye (1 bit).
            
            
            b. Non-Primitive Data Types
            
            - String: Characters ka sequence. Strings immutable hote hain.
            
            - Arrays: Same type ke multiple values store karne ke liye.
            
            - Classes: Custom data types, jo objects banate hain.
            
            
             5. Variables
            
            Declaration: Type ke saath variable ka naam define karna.
            
            int age; // Declaration
            age = 25; // Initialization
            
            
            Combined Declaration and Initialization:
            
            int age = 25; // Declaration and Initialization
            
            
            
             6. Operators
            Java mein different types ke operators hote hain:
            
            
            a. Arithmetic Operators
            
            + (Addition)
            
            - (Subtraction)
            
            \ (Multiplication)
            
            / (Division)
            
            % (Modulus)
            
            
             b. Relational Operators
            
            == (Equal to)
            
            != (Not equal to)
            
            > (Greater than)
            
            <(Less than)
            
            >= (Greater than or equal to)
            
             <=(Less than or equal to)
            
            
             c. Logical Operators
            - && (Logical AND)
            - || (Logical OR)
            - ! (Logical NOT)
            
            
            
             7. Control Structures
            
             a.Conditional Statements
            
            - if Statement:
            if (condition) {
                // code block if true
            }
            
            - if-else Statement:
            
            if (condition) {
                // code block if true
            } else {
                // code block if false
            }
            
            
            - switch Statement:
            
            switch (variable) {
                case value1:
                    // code block
                    break;
                case value2:
                    // code block
                    break;
                default:
                    // code block
            }
            
            
            
             b. Loops
            
            - for Loop:
            
            for (initialization; condition; update) {
                // code block
            }
            
            
            - while Loop:
            
            while (condition) {
                // code block
            }
            ```
            
            - do-while Loop:
            
            do {
                // code block
            } while (condition);
            ```
            
            
            8. Comments
            Java mein comments add karne ke liye:
            -Single Line Comment: // This is a comment`
            - Multi-Line Comment: /* This is a comment */`
            
            
            9. Input and Output
            
            - Scanner Class: User input lene ke liye.
            
            import java.util.Scanner;
            
            public class InputExample {
                public static void main(String[] args) {
                    Scanner scanner = new Scanner(System.in);
                    System.out.print("Enter your name: ");
                    String name = scanner.nextLine();
                    System.out.println("Hello, " + name);
                    scanner.close();
                }
            }
            ```
            
            
             10. Basic Exception Handling
            
            - try-catch Block: Errors ko handle karne ke liye.
            
            try {
                // code that may throw an exception
            }
             catch (ExceptionType e) {
                // exception handling code
            }
